Anleitung (README.md)
Ziel

Dieses Projekt stellt eine kleine Todo-Webapp bereit, die mit Docker Compose gestartet wird.
Die App läuft danach im Browser unter http://localhost:3000
.

Voraussetzungen (Lehrperson)

Docker Desktop installiert und gestartet

(Optional) Git installiert

Installation & Start (genau 3 Befehle)
git clone https://github.com/dms170800/uebungsprojekt
cd uebungsprojekt
docker-compose -f docker-compose/docker-compose.yml up -d

➡️ Danach im Browser öffnen:
http://localhost:3000
